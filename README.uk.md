# Автоматизована система класифікації травм на основі машинного навчання

[![English](https://img.shields.io/badge/lang-en-red.svg)](README.md)

---

Це репозиторій для коду дипломного проєкту на тему **"Автоматизована система класифікації травм на основі машинного навчання"**.

## Опис
Автоматизована система класифікації травм колінного суглоба за допомогою МРТ із використанням методів машинного навчання.

## Вимоги
### Апаратне забезпечення
- **GPU**: підтримка NVIDIA Automatic Mixed Precision (AMP)
- **VRAM**: мінімум 4 ГБ
- **RAM**: протестовано на 16 ГБ
### Software
- **Python 3.12+** (протестовано на 3.12.1)
- **CUDA driver** сумісний із вашим апаратним забезпеченням
- **Key Libraries**:
    - `torch` & `torchvision` (з підтримкою CUDA)
    - `opencv-python` (для попередньої обробки зображень)
    - `numpy`
    - `scikit-learn` (для метрик оцінювання)
    - `pillow`

## Key Features
- **Двоетапний пайплайн**: SСкринінг (бінарна класифікація) -> Діагностика (мультикласова класифікація).
- **Послідовне трансферне навчання (Transfer Learning)**: Тонке налаштування (fine-tuning) від бінарних ваг до спеціалізованого виявлення патологій.
- **Прискорення AMP**: Швидше навчання та зменшення споживання відеопам'яті (VRAM) завдяки змішаній точності.

## Configuration
Створіть файл `.env` у кореневій директорії проєкту та вкажіть шлях до вашого датасету:

`KNEE_CONDITIONS_DATASET=шлях/до/датасету/з/патологіями`\
`KNEE_DATASET=шлях/до/датасету/з/здоровими/зразками`
